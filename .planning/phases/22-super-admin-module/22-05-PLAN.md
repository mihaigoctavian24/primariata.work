- `status` filter: Apply `.eq("status", filters.status)` directly on the primarii query before assembly
- `search` filter: Apply `.ilike("nume_oficial", `%${filters.search}%`)` on the primarii query
- `judet` filter: Cannot filter at DB level (needs join); do post-assembly JS filter on judet field
- Same pattern for admins: status filter on user_primarii, search in JS post-assembly
