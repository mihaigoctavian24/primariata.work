**Role check helper (use inline in each action):**
```typescript
const supabase = await createClient();
const { data: { user } } = await supabase.auth.getUser();
if (!user) return { success: false, error: "Neautentificat" };
const { data: userData } = await supabase.from("utilizatori").select("rol").eq("id", user.id).single();

```

**getDashboardStats:** Uses `createServiceRoleClient()` (no await). Run these queries in parallel with `Promise.all`:
1. `supabase.from("primarii").select("id, status")` — count total, count active (status === "active")
2. `supabase.from("utilizatori").select("id", { count: "exact", head: true })` — total users
3. `supabase.from("cereri").select("id", { count: "exact", head: true })` — total cereri
4. `supabase.from("cereri").select("id", { count: "exact", head: true }).gte("created_at", firstDayOfCurrentMonth())` — cereri this month
5. `supabase.from("plati").select("suma").eq("status", "completed").gte("created_at", firstDayOfCurrentMonth())` — sum for MRR approximation

Return shape:
```typescript
interface DashboardStats {
  totalPrimarii: number;
  activePrimarii: number;
  totalUsers: number;
  totalCereri: number;
  cereriThisMonth: number;
  mrr: number;
}
```

Helper `firstDayOfCurrentMonth()`: `new Date(new Date().getFullYear(), new Date().getMonth(), 1).toISOString()`

Export signature: `export async function getDashboardStats(): Promise<{ success: boolean; data?: DashboardStats; error?: string }>`

**getPrimariiList:** Uses `createServiceRoleClient()`. Run in parallel:
1. `supabase.from("primarii").select("id, nome_oficial, status, created_at, localitate_id, config")` — all primarii
2. `supabase.from("localitati").select("id, nume, judet_id")` — all localitati
3. `supabase.from("judete").select("id, nume, abreviere")` — all judete
4. `supabase.from("user_primarii").select("primarie_id, user_id, rol, status")` — all associations
5. `supabase.from("utilizatori").select("id, prenume, nume, email")` — all users (for admin lookup)
6. `supabase.from("cereri").select("id, primarie_id, created_at")` — all cereri (for counts)
7. `supabase.from("plati").select("cerere_id, suma, status").eq("status", "completed")` — for revenue

Then assemble in JS: for each primarie, join localitate+judet, find admin user (user_primarii where rol IN ["admin","primar"] AND status="approved"), count usersCount, cereriTotal, cereriMonth, sum revenue.

Fields NOT in DB (show as null/undefined, UI must handle): uptime, satisfactionScore, avgResponseTime, cereriTrend, tier (read from `config?.tier ?? "Basic"`), features.
