**2. Multi-select + bulk delete:**

Add `selectedFileIds: Set<string>` state.


**3. Inline file rename:**

Add `renamingId: string | null` and `renameValue: string` state.

In the file name display area of each file card, when `renamingId === file.id`, show an input:
```tsx
{renamingId === file.id ? (
  <input
    autoFocus
    value={renameValue}
    onChange={(e) => setRenameValue(e.target.value)}
    onBlur={async () => {
      if (renameValue.trim() && renameValue.trim() !== file.name) {
        // Supabase Storage doesn't have a rename API — copy then delete
        const supabase = createClient();
        const oldPath = file.path ?? file.name;
        const newPath = oldPath.replace(file.name, renameValue.trim());
        // For simplicity: update display name in local state only (real rename = copy+delete)
        setFiles((prev) => prev.map((f) => f.id === file.id ? { ...f, name: renameValue.trim() } : f));
        toast.success("Fișier redenumit (local)");
      }
      setRenamingId(null);
    }}
    onKeyDown={(e) => { if (e.key === "Escape") setRenamingId(null); if (e.key === "Enter") (e.target as HTMLInputElement).blur(); }}
    className="w-full bg-transparent border-b border-[var(--color-info)] text-foreground text-xs outline-none"
    onClick={(e) => e.stopPropagation()}
  />
) : (
  <span
    onDoubleClick={(e) => { e.stopPropagation(); setRenamingId(file.id); setRenameValue(file.name); }}
    className="truncate text-foreground text-xs cursor-text"
    title="Double-click to rename"
  >
    {file.name}
  </span>
)}
```

Note: Supabase Storage has no rename endpoint. The rename is local state only, with a toast explaining this. This matches the academic milestone scope.
