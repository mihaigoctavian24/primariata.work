**Step B — Enhance utilizatori-content.tsx:**

1. **Sort state**: Add `sortField: "name" | "rol" | "status" | "last_login"` and `sortDir: "asc" | "desc"` state (default: "rol" / "desc").

2. **Sortable column headers**: In the `<thead>`, wrap each column label in a button:
   ```tsx
   <button onClick={() => handleSort("name")} className="flex items-center gap-1 text-muted-foreground hover:text-foreground text-xs font-semibold uppercase tracking-wider transition-colors cursor-pointer">
     Nume
     {sortField === "name" ? (
       sortDir === "asc" ? <ArrowUp className="w-3 h-3" /> : <ArrowDown className="w-3 h-3" />
     ) : (
       <ArrowUpDown className="w-3 h-3 opacity-30" />
     )}
   </button>
   ```
   Import `ArrowUp, ArrowDown, ArrowUpDown` from lucide-react.
   `handleSort` toggles dir if same field, otherwise resets to desc.
   Apply sort in the `useMemo` filtered list: sort by `sortField` after existing filters.

3. **Bulk selection**: Add `selectedIds: Set<string>` state.
   - Add checkbox as first column in each row: `<input type="checkbox" checked={selectedIds.has(user.id)} onChange={...} className="accent-[var(--color-info)] cursor-pointer" />`
   - Add header checkbox for select-all (checked when selectedIds.size === paginated.length)
   - When `selectedIds.size > 0`, render BatchActionBar above table:
     ```
     [N utilizatori selectați] [Activează] [Suspendă] [Deselectează]
     ```
     - "Activează" calls `updateUserStatus` for each selected id with status "activ", then clears selection
     - "Suspendă" calls `updateUserStatus` for each with status "suspendat"
     - Import `updateUserStatus` from `./actions`

4. **Advanced filter button**: Add a "Filtre Avansate" button (Filter icon) next to the search bar.
   Clicking sets `showAdvancedFilter: true`.
   Render `<AdvancedFilterModal open={showAdvancedFilter} onClose={...} filters={advancedFilters} onApply={(f) => { setAdvancedFilters(f); setPage(1); setShowAdvancedFilter(false); }} departamente={uniqueDepartamente} />`
   Where `uniqueDepartamente` is computed from the user list: `Array.from(new Set(users.map(u => u.departament).filter(Boolean)))`.
   Apply `advancedFilters` in the useMemo: after existing role tab filter, also filter by advancedFilters.rol/status/departament.

5. **CSS token status colors**: Replace the hardcoded `statusColor` helper:
   ```typescript
   function statusColor(status: string | null): string {
     const colors: Record<string, string> = {
       activ: "bg-[var(--color-success-subtle)] text-[var(--color-success)]",
       suspendat: "bg-[var(--color-error-subtle)] text-[var(--color-error)]",
       pending: "bg-[var(--color-warning-subtle)] text-[var(--color-warning)]",
       inactiv: "bg-[var(--color-neutral-subtle)] text-[var(--color-neutral)]",
     };
     return colors[status ?? ""] ?? "bg-[var(--color-neutral-subtle)] text-[var(--color-neutral)]";
   }
   ```
