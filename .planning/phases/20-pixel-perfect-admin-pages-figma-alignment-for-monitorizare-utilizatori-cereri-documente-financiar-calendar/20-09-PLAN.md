**Step A — Update src/store/calendar-store.ts:**

The existing `CalEvent` interface needs two new optional fields:
```typescript
export interface CalEvent {
  id: string;
  title: string;
  date: string;       // ISO string YYYY-MM-DD (existing)
  time: string;
  color: string;      // Tailwind bg-* class (existing)
  type: string;       // Event type label (existing, now populated)
  location?: string;
  recurrence?: "none" | "zilnic" | "saptamanal" | "lunar"; // NEW
}
```

Add a new `updateEvent` action to the store (alongside existing `addEvent` / `removeEvent`):
```typescript
updateEvent: (id: string, updates: Partial<Omit<CalEvent, "id">>) => void;
```

Implementation in `create`:
```typescript
updateEvent: (id, updates) =>
  set((state) => ({
    events: state.events.map((e) => (e.id === id ? { ...e, ...updates } : e)),
  })),
```
