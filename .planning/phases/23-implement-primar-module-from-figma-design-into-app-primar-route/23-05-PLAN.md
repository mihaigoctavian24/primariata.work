**Approve action:**
```typescript
const [isPending, startTransition] = useTransition();
const router = useRouter();

async function handleApprove(cerereId: string) {
  startTransition(async () => {
    const result = await approveCerere(cerereId);
    if (!result.error) {
      confetti({ particleCount: 100, spread: 70, origin: { y: 0.6 } });
      router.refresh();
    } else {
      // show inline error on the card
    }
  });
}
```

**Reject action:** calls rejectCerere(cerereId). router.refresh() on success. No confetti.

**Note action:** inline text input expands below the card (useState toggleNoteId). On submit: addNotaPrimar(cerereId, noteText). router.refresh() on success.
