**List view** (default):
Each ProiectRow displayed as a card:
- Project name (font-semibold) + responsabil (text-gray-400 text-sm)
- Status badge (color-coded pill):
  - in_derulare: `bg-blue-500/10 text-blue-400 border border-blue-500/20`
  - planificat: `bg-gray-500/10 text-gray-400 border border-gray-500/20`
  - finalizat: `bg-emerald-500/10 text-emerald-400 border border-emerald-500/20`
  - intarziat: `bg-red-500/10 text-red-400 border border-red-500/20`
- Progress bar: `{progres_pct}%` label + `h-1.5 rounded-full bg-white/[0.06]` with amber fill
- Budget: `{buget_consumat.toLocaleString("ro-RO")} / {buget.toLocaleString("ro-RO")} RON`
- Deadline: deadline date formatted as ro-RO
- Edit button (Pencil icon) + Delete button (Trash2 icon)

**Create form** — shown as an inline expanded card at top of list (toggleable via "Proiect nou" button):
Fields:
- nume (text input, required)
- categorie (text input, optional)
- status (select: in_derulare | planificat | finalizat | intarziat)
- progres_pct (number input 0-100)
- buget (number input)
- buget_consumat (number input)
- deadline (date input)
- responsabil (text input)

Submit calls createProiect(). On success: router.refresh() + collapse form. On error: inline error text below form.

**Edit modal** — simple inline expansion (same fields as create but pre-populated). Opens when Pencil clicked. Calls updateProiect(id, data) on submit.

**Delete confirm** — inline "Ești sigur?" row below the project card (not a separate modal — keeps it simple). "Da, șterge" calls deleteProiect(id) + router.refresh(). "Anulează" dismisses.

All actions use `useTransition` to show pending state (buttons disabled during transition).

Pending state: buttons show a spinner (or just `opacity-50 pointer-events-none`).

**Create button** at top right: "Proiect nou" with Plus icon, amber gradient background.

Empty state: "Nu există proiecte" with Landmark icon and "Proiect nou" CTA button.
