# Plan 03: Settings Page Visual Overhaul to Match Figma Reference

**Goal**: Restyle all 5 admin settings tab components to match the Figma reference design (Revamp Primarie Admin/src/app/components/pages/SetariPage.tsx). Fix tab switching speed issue.

**Reference**: `Revamp Primarie Admin/src/app/components/pages/SetariPage.tsx`

## Design Tokens from Figma Reference

```
- Avatar: background linear-gradient(135deg, #8b5cf6, #6366f1), rounded-2xl, 80x80
- Input wrapper: background rgba(255,255,255,0.04), border 1px solid rgba(255,255,255,0.06), rounded-xl, px-3 py-2.5
- Input icon: w-4 h-4 text-gray-600, inside input wrapper before the text
- Save button: background linear-gradient(135deg, #ec4899, #8b5cf6), rounded-xl, with Save icon
- Tab indicator: background linear-gradient(135deg, rgba(236,72,153,0.12), rgba(139,92,246,0.06)), border 1px solid rgba(236,72,153,0.12)
- Content card: background rgba(255,255,255,0.025), border 1px solid rgba(255,255,255,0.05), rounded-2xl, p-6
- Toggle: custom animated spring toggle with gradient background when active
- Font sizes: labels 0.8rem, inputs 0.88rem, headings 1.05rem
```

## Tasks

### Task 1: Fix Tab Switching Speed + Page Header + Tab Indicator Styling
**Files**: `admin-settings-tabs.tsx`, `page.tsx`
**Changes**:
1. **page.tsx**: Add Settings (lucide) icon before "Setari" heading
2. **admin-settings-tabs.tsx**: Change `AnimatePresence mode="wait"` to `mode="popLayout"` with shorter transition (100ms vs 200ms) to eliminate perceived lag
3. **admin-settings-tabs.tsx**: Update tab indicator background to use Figma gradient: `linear-gradient(135deg, rgba(236,72,153,0.12), rgba(139,92,246,0.06))` with border `1px solid rgba(236,72,153,0.12)` instead of current accent tint
4. **admin-settings-tabs.tsx**: Increase tab button padding/size to match Figma (py-3, rounded-xl, font 0.88rem)

### Task 2: Create Shared InputWithIcon Component + Gradient Save Button Pattern
**Files**: New `src/components/admin/settings/settings-ui.tsx`
**Changes**:
1. Create `InputWithIcon` component - wraps input field with an icon on the left inside a styled container matching Figma:
   - Container: `bg-white/[0.04] border border-white/[0.06] rounded-xl px-3 py-2.5 flex items-center gap-2`
   - Icon: `w-4 h-4 text-gray-600` (passed as prop)
   - Input: `bg-transparent text-foreground outline-none flex-1 text-[0.88rem]`
   - Supports readonly, placeholder, error state
   - Works with react-hook-form register()
2. Create `GradientSaveButton` component:
   - Style: `background: linear-gradient(135deg, #ec4899, #8b5cf6)`, rounded-xl, text-white
   - Includes Save icon from lucide
   - Supports loading state with Loader2 spinner
   - whileHover scale 1.02, whileTap scale 0.98 (from Figma)
3. Create `AnimatedToggle` component matching Figma toggle style:
   - Custom button-based toggle with spring animation for the knob
   - Active: gradient background `linear-gradient(135deg, #ec4899, #8b5cf6)`
   - Inactive: `rgba(255,255,255,0.1)` background
   - Spring transition stiffness 500, damping 30
